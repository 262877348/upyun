  • CDN 加速：

    • 文件访问方式：http://绑定域名/文件路径
    • 例如： 服务名 bucket文件路径 /dir/pic.jpg测试域名 test_domain用户绑定域名 example.yourdomain.com
    • 该图片对外访问地址为： http://test_domain/dir/pic.jpghttp://example.yourdomain.com/dir/pic.jpg

    直播加速：

    • 直播推流地址：协议://推流域名/接入点/流名   直播播放地址：协议://播放域名/接入点/流名
    • 例如： 服务名 bucket接入点 app流名 stream推流域名 CNAME bucket.s1.aicdn.com用户绑定推流域名 push.domain.com播放域名 CNAME bucket.s0.aicdn.com用户绑定播放域名 pull.domain.com
    • rtmp 推流地址为： rtmp://push.domain.com/app/stream
    • rtmp 播放地址为： rtmp://pull.domain.com/app/stream
    • http-flv 播放地址为： http://pull.domain.com/app/stream.flv
    • hls 播放地址为： http://pull.domain.com/app/stream.m3u8

  • FTP 登录方法：

    • 工具：Windows 用户推荐客户端 FileZilla、WinScp
    • 地址（主机）： v0.ftp.upyun.com (智能选路)v1.ftp.upyun.com (电信线路)v2.ftp.upyun.com (联通线路)v3.ftp.upyun.com (移动线路)
    • 加密：选择“不加密”使用 FTP，选择“显式的 FTP over TLS”使用 FTPS
    • 用户名：操作员名/服务名，如 operator/mybucket（需要两个同时填写）
    • 密码：操作员的密码
    • 端口：21
